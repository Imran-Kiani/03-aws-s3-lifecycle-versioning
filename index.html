About This Project

This project is a high-performance, production-grade web architecture designed to demonstrate the power of Infrastructure as Code (IaC) and Automated Deployment. By leveraging the AWS Global Infrastructure, this site ensures near-zero latency and high availability for users worldwide.

The entire environment was provisioned using Terraform, ensuring that the infrastructure is version-controlled, repeatable, and scalable. This represents a modern approach to cloud management where manual configuration is replaced by precise, executable code.

Deployment Status: ● Live via S3 & CloudFront OAC

Technical Stack

AWS CloudFront (CDN) Amazon S3 Terraform (IaC) GitHub Actions (CI/CD) IAM & OAC Security Route 53 & SSL

Automated CI/CD Pipeline (The Power of DevOps)

This website is integrated with a sophisticated Continuous Integration and Continuous Deployment (CI/CD) pipeline. The main advantage of this setup is the elimination of manual uploads and human error.

With GitHub Actions, any change I make to the code is automatically tested and synced to the AWS Cloud. I can update my entire global web presence with just three simple commands:

  • 1. git add .
  • 2. git commit -m "Update website"
  • 3. git push origin main

Once pushed, the pipeline handles the rest—invalidating the CloudFront cache and serving the fresh content to users globally within seconds.

Key Architecture Benefits

  • Security: Origin Access Control (OAC) ensures that the S3 bucket is completely private, making CloudFront the only gateway.
  • Performance: Content is cached at AWS Edge locations worldwide, reducing load times significantly.
  • Cost-Efficiency: Serverless architecture ensures that the cost is only incurred based on actual traffic.
